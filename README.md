# ETHARA PROJECT

ETHARA is a full-stack task and project management web application designed for admins and students. It allows admins to create and assign tasks/projects, while students can log in, view assigned work, and submit their work through links.

## 🌐 Live Demo

Frontend: https://ethara-project-five.vercel.app  
Backend API: https://ethara-backend-3i2t.onrender.com

---

## 🚀 Features

### Admin Panel
- Secure Login / Registration
- Create Tasks
- Create Projects
- Assign tasks/projects to students
- Set Deadlines
- View submissions

### Student Panel
- Secure Login / Registration
- View Assigned Tasks
- View Assigned Projects
- Submit Task Links
- Submit Project Links
- Check Deadlines
- Status Tracking
